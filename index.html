About this Project

This project incorporates job data pulled from Indeed. Using data tools like Python, SQL, HTML/CSS/JavaScript, we were about to find trends in the job market for Catalina Island.

Step 1

We used Beautiful Soup and Python as tools to webscrape the Indeed website.

Step 2

Once a Pandas dataframe has been created using the cleaned data, we utilized SQL and created new tables.

Step 3

After we extracted the cleaned SQL data, we created HTML tables to display the information.

Step 4

At the end, we used D3 from Javascript and GEOJSON to create map markers.
